Route 19
Bulletproof Lobby Kiosk System

Requirements

  • Lenovo M75q-1 Tiny (or compatible hardware)
  • Boot from Arch Linux ISO
  • Internet connection (Ethernet recommended)
  • ⚠️ All data on target disk will be erased

One-Command Installation

Boot into the Arch Linux ISO and run:

curl -sSL https://raw.githubusercontent.com/kenzie/lobby-arch/main/arch-install.sh -o /tmp/arch-install.sh chmod +x /tmp/arch-install.sh /tmp/arch-install.sh
Warning: This will completely wipe the selected disk and install a fresh Arch Linux system configured for lobby display use.

Installation Steps

1 Boot Arch ISO: Insert USB and boot target machine
2 Connect Internet: Ensure ethernet is connected
3 Run Installer: Copy and paste the command above
4 Follow Prompts: Select disk and set password
5 Reboot: System configures during installation and boots directly into working kiosk
